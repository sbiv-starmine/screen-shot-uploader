# screen-shot-uploader

## Getting Started

### Firebase configuration
Fill ```ApiKeys.js```.
```bash
$ mv ApiKeys.sample.js ApiKeys.js
```

### Running the Expo CLI server
```bash
$ npm install
$ npm start
```
The npm start command starts the Expo CLI server.
Make sure you have Expo-CLI installed. 

## Overview
- React Native Expo  
- Firebase
